1. Ući u 'trunk' folder

2. Pokrenuti sljedeće naredbe iz Command Prompt-a:

- npm install ............................ (Instalira sve npm module potrebne za Gulp)
- gulp install ........................... (Download ionic i angular javascript file-ova)
- ionic platform add android ............. (Instalira android platformu i sve plugine definirane u package.json file-u)
- ionic build android [--production] ..... (Builda applikaciju za mobilni uređaj i prije toga izvršava Gulp taskove za minificiranje koda.)
- ionic run android [--production] ....... (Builda applikaciju za mobilni uređaj i prije toga izvršava Gulp taskove za minificiranje koda. 
                                            Zatim instalira aplikaciju na mobilni uređaj. Mobilni uređaj mora biti spojen kabelom na računalo)
- ionic serve [--nolivereload] ........... (Pokreće aplikaciju u browseru. Gulp watch task se izvršava na svaku promjenu u kodu dok je naredba aktivna. 
                                            Ako nije postavljen --nolivereload flag Gulp watch ne može pratiti promjene)

OPCIONALNO

3. Prijaviti aplikaciju na ionic.io
- ionic io init .......................... (Generira app ID i api KEY i prijavljuje aplikaciju na https://apps.ionic.io/apps stranicu)

4. Instalirati Ionic core servis
- npm install -g bower ................... (Uvjeriti se da je bower instaliran)
- ionic add ionic-service-core ........... (Instalira core service modul potreban za rad svih ostalih ionic service modula.
                                            NAPOMENA: Naredba sama ubacuje javascript snippet u index.html file)

5. Push notifikacije
- npm update -g cordova ................................................. (Uvjeriti se da je zadnja verzija cordove instalirana)
- ionic plugin add https://github.com/phonegap-build/PushPlugin.git ..... (Instalira push plugin)
- ionic add ngCordova ................................................... (Instalira wrapper za ostale cordova plugine
                                                                           NAPOMENA: Naredba sama ubacuje javascript snippet u index.html file)
- ionic add ionic-service-push .......................................... (Instalira ionic push modul
                                                                           NAPOMENA: Naredba sama ubacuje javascript snippet u index.html file)
- ionic config set dev_push true ........................................ (Dozvoljava development push da bi se uvjerili da je push dobro.
                                                                           konfiguriran. Koristiti samo za provjeru push funkcionalnosti.
                                                                           NAPOMENA: Naredba sama ubacuje javascript snippet u index.html file)

- dodati potreban javascript kod u projekt 	- http://docs.ionic.io/docs/push-from-scratch
- setupirati Android GCM servis 		- http://docs.ionic.io/docs/push-android-setup

- ionic push -s ......................................................... (Testira push notifikacije. Potreban device TOKEN i secret api KEY)

6. Deploy
- ionic add ionic-service-deploy ........................................ (Instalira Ionic deploy servis
                                                                           NAPOMENA: Naredba sama ubacuje javascript snippet u index.html file)
- ionic plugin add ionic-plugin-deploy .................................. (Instalira deploy plugin)

7. Analytics
- ionic add ionic-service-analytics ..................................... (Instalira Ionic analytics servis
                                                                           NAPOMENA: Naredba sama ubacuje javascript snippet u index.html file)
